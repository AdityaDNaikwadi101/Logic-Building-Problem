# Problems on Digits

This module contains **digit-based logic programs** that operate on individual digits of a number.  
The objective is to build a strong understanding of **digit extraction and digit-wise processing**.

---

## Problem Scope

The following problems are implemented in this section:

- Display digits of a number
- Count total digits
- Count even digits
- Count odd digits
- Calculate sum of digits
- Calculate sum of even digits
- Calculate sum of odd digits
- Reverse a number
